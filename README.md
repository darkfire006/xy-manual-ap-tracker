# Pokemon-XY-Manual-Poptracker
Poptracker Pack for the Pokemon XY Manual by Snifit!

This Package is made for use in [Poptracker](https://github.com/black-sliver/PopTracker) for the Pokémon XY Manual [Archipelago Randomizer](https://archipelago.gg/) created by [Snifit](https://github.com/Snifit57). That Project can be found [Here!](https://github.com/Snifit57/PokemonXYAPManual) 

This Project was made using the Platinum Manual tracker as a base, you can find that [here.](https://github.com/ZobeePlays/Pokemon-Platinum-Manual-Poptracker)

Usage:
Connecting this tracker to your AP slot will act as a replacment to the manual client, as where clicking off checks in the tracker will send out the corresponding check to your ap room!
